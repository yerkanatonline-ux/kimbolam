# Мотивациялық эссе көмекшісі — прототип

## Файлдар
- `index.html` — прототип беті (чат + эссе жоба тақтасы, екі бөлек панель)
- `../backend/api/essay-coach.js` — backend endpoint (AI кеңесшімен бір Vercel жобасында орналасады)

## Екі режим
1. **Coach (әдепкі)** — AI толық эссе жазбайды, тек бағыттаушы сұрақ қояды және
   студенттің өз жобасына («Кері байланыс сұра» батырмасы арқылы) кері байланыс береді.
2. **Full draft (тек сұрағанда)** — «Толық үлгі сұра» батырмасы басылғанда ғана іске
   қосылады, әрдайым «бұл тек бастапқы нүкте, көшірме» деген ескертумен бірге жазады.

## Деплой
`backend/` қалтасында `essay-coach.js` файлы `chat.js`-пен бірге автоматты түрде
бір Vercel жобасына енеді (қосымша деплой қадамы қажет емес, себебі бір `api/` қалтасында).
Frontend-тегі `AI_API_URL`-ды нақты деплой URL-іне ауыстыру керек (`/api/essay-coach`).

## Ескерту
Тест профилі (`profile` айнымалысы) қазір қолмен қойылған (ISA/INFP). Негізгі
платформаға қосылғанда бұл `aiProfile`-дан (showResults() толтыратын) автоматты алынады.
