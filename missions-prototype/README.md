# IDP миссиялары — прототип

Үлкен идея: тест бір реттік сурет («сен кімсің»), ал миссиялар — уақыт бойы
жалғасатын даму жүйесі («сен қайда барасың»). 5 деңгей (кітаптың бөлімдеріне сай):
Тану → Мақсат қою → Жоспарлау → Әрекет ету → Бағалау.

## Осы қалтада
- `mission-1-goal.html` — бірінші миссия (Мақсат қою), толық айналатын прототип:
  кіріспе → AI серігімен диалог (мақсатты нақтылау) → мақсат карточкасын толтыру →
  бейдж/XP аяқталу мерекесі.
- `../backend/api/goal-mission.js` — backend (AI кеңесшімен бір Vercel жобасында).

## Маңызды принцип
AI студенттің орнына мақсат қоймайды — тек нақтылаушы сұрақ қояды («қашан?»,
«қалай өлшейсің?», «неге маңызды?»). Дәл эссе көмекшісіндегі «coach, ghostwriter
емес» принципі осында да қолданылған.

## Әлі істелмегені
- Мақсат әлі дерекқорға сақталмайды (консольге ғана шығады) — save-result
  паттерні бойынша жаңа `goals` кестесі мен endpoint керек болады.
- Қалған 4 деңгей (Тану деп есептеледі, Жоспарлау/Әрекет ету/Бағалау) әлі жоқ.
- Робот кейіпкері дауыстық кіріспедегімен бірдей дизайнда, бірақ әлі ортақ
  компонентке бөлінбеген (екі файлда қайталанып тұр).
